# Karuvadukadai WhatsApp AI Bot

This project connects **Interakt WhatsApp API** with **OpenAI (ChatGPT)** to create an automated WhatsApp chatbot for customer support.

The bot:
- Receives WhatsApp messages via Interakt Webhook  
- Sends the message to OpenAI  
- Gets an AI-generated reply  
- Sends the reply back to the customer on WhatsApp
